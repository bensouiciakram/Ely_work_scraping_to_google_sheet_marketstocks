How to run this code in your local, for debug, for testing:

First run:

1. Create virtual enviroment

2. Run

   `python3 or python save_stocks.py`

This will take a lot of times, maybe your computer will be heated up, so you can use the example-stocks.json

- Copy the example-stocks.jso
- Create a stocks.json
- Paste into stocks.json

Then you ready to go

3. Run this if first time

   `playwright install`

4. Create a Google Service Account, paste the json file to auth folder, inside notification-88f1b-7105c5bbca71.json

5. Create your own new google sheet, note down the id, then open that google sheet, click share, then allow the created Google Service Account above to use that google sheet, if not clear this step, ask ChatGPT

6. Change sheet tab name to: Stock Price & DCF Websites

7. Run the scrape_details.py and see the result,

8. Please document anything you do want next developer to understand and help everybody to easy to maintain your code. Thank you
